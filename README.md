Projet AlgoBox – TP CN Numérique

Ce dépôt contient le travail réalisé dans le cadre du TP guidé portant sur :
- la prise en main d’AlgoBox,
- l’utilisation de Git et GitHub,
- la collaboration (branches, pull, merge, résolution de conflits),
- la mise en place d’un rendu structuré.

Contenu du dépôt

### `src/`
Ce dossier contient les fichiers sources du projet, notamment :

- **TP_CN.algo**  
  Le fichier AlgoBox final utilisé pour le rendu.

Comment ouvrir et exécuter le fichier AlgoBox

1. Ouvrir le logiciel **AlgoBox**
2. Faire : **Fichier → Ouvrir**
3. Sélectionner le fichier :  
   `src/TP_CN.algo`
4. Lancer l’algorithme avec le bouton **Exécuter**

Le programme affichera les résultats dans la fenêtre d’exécution d’AlgoBox.


Organisation Git du projet

Le dépôt montre l’utilisation de différentes fonctionnalités Git :

- Création de branches (`main`, `partenaire`)
- Modifications successives
- Commits réguliers
- Merge entre branches
- Résolution de conflits
- Push et pull via GitHub
- Historique consultable dans l’onglet **Commits**

Lien GitHub figé (tag)
